Basic Positions

  • Guard

    • Used to control an opponent from your back with both legs locked around opponent

  • Half Guard

    • Used to control an opponent from you back with one leg captured

  • Butterfly Guard

    • Used to control an opponent with both feet hooked under opponents legs

  • Mount

    • Used to control an opponent when on top

  • Side Control

    • Used to control an opponent when on the side

  • Knee on belly

    • Used to control and opponent when on top with one knee pinning opponent's stomach

  • Back Control

    • Used to control an opponent when sitting behind an opponent with both feet hooking opponent's legs
